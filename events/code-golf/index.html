Welcome to

Code Golf!

The rules: solve the question with the shortest amount of characters to win. The team with the most questions solved at the end of the match wins. Ties will be broken by submission time.

For each question, write code in whatever IDE or editor you'd like. You can use any of the following languages: Bash, C, C++, C#, Haskell, Java, JavaScript, Lua, Pascal, Perl, PHP, Python, R, Racket, Ruby, or Rust. Write your code in a function that prints the output. If the language you're using requires extra boilerplate code, you don't need to include it, just the function definition. For libraries, you should only use the ones that come standard with your language, no third-party that require installation.

To submit your code, click the "Submit" button and upload the code file (for python, .py, for Java, .java, etc). Any member of your team can submit answers for your team. Just be sure to include your team number on the form.

1. Double Up

3. Leap Years

A leap year is a year which is divisible by 4, except for centennial years, unless they are divisible by 400.

Write a program that prints out all leap years from 1800 to 2400, each one on a new line.

Hints

The years 
1804, 1808, 1812, 2400
are leap years.
The years 
1800, 1900
are not.

4. Out Loud

Example 1

zero


Example 2

Example 3

Input: 
521

5. Generate a Plate

Given an abbreviation of a U.S. state as an input, randomly generate a valid license plate number in that state.

A full list of plate formats in each state can be found here. Each 
A
can be any letter A-Z, and each 
0
can be any number 0-9. Comma-separated lists can be any of the listed formats.

Example 1

Input: 
"TX"

Example output: 
SCV-1924


Example 2

Input: 
"IN"

Example output: 
824XR


Example 3

Input: 
"UT"

Example output: 
H82 7EW
